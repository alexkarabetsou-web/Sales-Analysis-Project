# 📊 Sales Analysis Project

## Overview
This project explores sales data using Python, Pandas and Matplotlib.

The goal was to clean the dataset, perform exploratory data analysis (EDA), and answer business-related questions through data visualization.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed

### 1. Data Cleaning
- Removed missing values
- Converted date columns to datetime format
- Created additional time-based features

### 2. Sales by City
- Compared revenue across different cities
- Identified top-performing locations

### 3. Orders by Hour
- Analyzed customer purchasing behavior throughout the day
- Identified peak ordering periods

### 4. Products Bought Together
- Used order IDs to identify products frequently purchased in the same transaction

### 5. Product Performance
- Compared product sales and popularity

## Key Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- Business Insights Generation
